Abra o painel de controle Xampp
Clique em 'config' atrás de 'Apache'
Selecione 'PHP (php.ini)' no menu suspenso -> Um arquivo agora deve abrir no seu editor de texto
Pressione ctrl + f e procure "max_execution_time", você deve refinar uma linha que apenas 

max_execution_time = 30

Mude 30 para um número maior (180 trabalharam para mim), assim:

max_execution_time = 180
Salve o arquivo
Servidor 'Stop' Apache
Fechar Xampp
Reiniciar o Xampp
'Iniciar' Servidor Apache
Atualize o Wordpress do painel de administração
Apreciar ;)
